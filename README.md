# Networking Portfolio

This repository contains structured notes, blogs, and lab-based troubleshooting
scenarios created while studying switching, VLANs, and wireless networking.

## Focus Areas
- VLANs and 802.1Q trunking
- Native VLAN behavior and risks
- Inter-VLAN routing using SVIs
- Practical troubleshooting using CLI commands

## Tools Used
- Cisco IOS (Packet Tracer)
- GitHub Pages
- Markdown
